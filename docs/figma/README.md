# Figma-Related Files

This folder contains files that were generated by Figma Make during the initial project conversion from Figma design to React code.

## Files in this folder:

### ATTRIBUTIONS.md
- Attribution file for shadcn/ui and Unsplash images
- Required for license compliance

### default_shadcn_theme.css
- Default theme CSS from Figma Make's shadcn integration
- Reference file for the original design system tokens
- Not actively used in production

### pnpm-workspace.yaml
- Workspace configuration from Figma Make
- Not needed for the current project structure

### PROJECT_SUMMARY.md
- Original project summary generated during Figma → React conversion
- Historical reference document
